User Stories from Product Backlog that we will address in this sprint:
a) As a terminal user it would be nice to keep track of previous folders so I can access them later
b) As a terminal user I would like to be able to change into a previous folder saved within my terminal
c) As a terminal user I want to be able to quickly make files
d) As a terminal user I want to be able to overwrite files when I desire
e) As a terminal user I would like to be able to add to files whenever I want
f) As a terminal user I would like to see if something went wrong with the program
g) As a terminal user I would like a visual way to interact with the file system
